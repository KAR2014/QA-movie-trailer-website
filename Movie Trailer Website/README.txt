README
------------
To run the program ensure that you have Python 2.7 installed on your system.
Run the "entertainment_center.py" file so that the rotten_tomatoes HTML page is generated

Information on the files:
media.py: contains the Movies class which is used to construct the individual movie objects that are shown on the HTML page
entertainment_center: contains the movie objects, movie list and the object that creates the HTML page
fresh_tomatoes: generates the HTML page
